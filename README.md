### Install
```git clone https://github.com/blotgg/btwnode.git && cd btwnode && bash install.sh```

### Start
```sudo docker-compose up -d```

### Stop
```sudo docker-compose stop```

### Logs
```tailf -n 10 logs/debug.log```
Ctrl+C - exit log mode
